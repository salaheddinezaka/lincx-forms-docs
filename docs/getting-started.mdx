# Getting Started with Lincx Forms Builder

A Quick overview of how Lincx Forms Works

## What is Form Builder?

Form Builder is a solution to build lead forms, it provides both single-page forms and multi-pages form.

This tool will help you create powerful multi-step forms to capture up to 5X more leads. Capture, verify & submit your data to any kind of lead system you'd like
